com.bnorm.junit5.contingent.Contingent

Marks the current Test as being contingent on the success of other tests. Tests are matched using values specified by this annotation against other tests with equal Tag values. If any test with any matching Tag fails, this test will be failed.

com.bnorm.junit5.contingent.ContingentExtension

Class-level extension annotation which must be applied for Contingent annotations to be used.
